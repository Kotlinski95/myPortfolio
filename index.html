// PLC programmer carrier

Adrian Kotliński, 27.10.2020

PLC programming vs Front-End

How to change profession from Automatic area into Front-End? It's not easy road, but there are similarities in both profession...

In my case, carrier in automotive area was based on programming Programable Logic Controllers. There are much device to control: servomechanisms, inverters, safety devices, RFID systems etc. I used high level programming languages ​​like SCL, ST, C, C ++ in PLC programming, so I'm familiar with the structure of the JavaScript language. Each automation system has a visualization system, so designing the appearance of web applications doesn't much differ from scada systems in car factories. I participated in several automotive projects, where we programmed production lines for such automotive elements as cockpit assembly, body welding and engine transport. Working as an automation specialist taught me resistance to stress, strict adherence to deadlines, reliability and responsibility for the work entrusted to me
